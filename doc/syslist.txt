 List is taken from syscalls(2).                                      
                                                                      
 The list of system calls that are available as at kernel 4.5 (or in a
 few cases only on older kernels) is as follows:                      

 Tag list:

 fork           - clone, fork, vfork
 tthread        - control execution of this particular thread, excluding fork & exec.
 file (+ epoll) - operations on a single file (possibly with epoll)
 fs             - operations on fs (mkdir, etc)
 sys            - all operations that change or get system state
 sock           - operations on sockets
 mem            - operations with memory
 exec           - exec
 pipe           - pipes
 ipc            - communications between processes
 futex          - futexes
